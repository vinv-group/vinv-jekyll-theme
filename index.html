vinv standardizes forest inventory data enabling automated processing for visualization, analysis and forest management.

" class="headline-module-left" img="/assets/img/oc100_logo.jpg" %} {% include section-hero_right.html subtitle="Big Data" title="Precision Forestry" text="

The vinv exchange format is made for individual tree data by default.

The format supports the results of upcoming inventory methods such as terrestrial or airborne laser scanning.

Machine learning models can also be incorporated to make forest stand prediction.

" class="headline-module-right" img="/assets/img/oc100_logo.jpg" %} {% include section-hero.html subtitle="Open Source" title="Driven by Community" text="

The software is and will forever be available for free.

Open development from the beginning with the involvement of all interested parties.
